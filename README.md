# Harv Balu – REALTOR® Landing Page

A single-page website for **Harv Balu**, REALTOR® serving Fremont, Newark, Milpitas, Union City & the Bay Area. Data-driven insights and personalized service.

## What’s in this repo

- **index.html** – Static landing page (HTML, CSS, and JS in one file)
- **.gitignore** – Ignores macOS and temp files

No build step or dependencies. Open `index.html` in a browser to view locally.

## Run locally

1. Clone the repo (or download and unzip).
2. Open `index.html` in your browser (double-click or **File → Open**).

Or from the project folder:

```bash
open index.html
```

## Publish with GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Branch: **main**, folder: **/ (root)**.
5. Save. Your site will be at `https://<username>.github.io/<repo-name>/`.

## Tech

- Plain HTML5, CSS3, JavaScript
- Responsive layout, semantic markup, basic accessibility (ARIA where needed)
- External assets over HTTPS (fonts, images)
